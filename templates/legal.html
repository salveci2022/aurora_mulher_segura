📜 TERMOS DE USO

1. Finalidade
O Aurora Mulher Segura é um sistema de emergência desenvolvido para auxiliar mulheres em situações de risco. Não substitui serviços oficiais de emergência.

2. Dados coletados
O sistema coleta: nome (opcional), tipo de situação, mensagem (opcional) e localização GPS (somente se autorizada pela usuária). Os dados são armazenados localmente no servidor.

3. Localização GPS
A localização só é capturada e transmitida com consentimento explícito da usuária. O dado é usado exclusivamente para envio às pessoas de confiança cadastradas.

4. Responsabilidade
O Aurora é uma ferramenta auxiliar. Em emergências, sempre acione os serviços oficiais: Polícia (190), SAMU (192) ou Central da Mulher (180).

5. Privacidade
Os dados não são compartilhados com terceiros. As senhas são armazenadas de forma criptografada. Os alertas ficam visíveis apenas para o admin e pessoas de confiança cadastradas.
